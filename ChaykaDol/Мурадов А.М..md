# Информационная система «Чайка» (ChaykaDol)

**Разработчик:** Мурадов Амал Мустаджабович  
**Группа:** ИСП(с)3223  
**Технологии:** C#, WPF, SQL Server, Entity Framework Core

## Описание проекта
Информационная система предназначена для автоматизации учёта деятельности ДОЛ «Чайка» (г. Уфа).

## Функциональные возможности
- Учёт детей (добавление, удаление, поиск по ФИО)
- Учёт отрядов и их распределение по корпусам
- Ведение медицинских заявок
- Планирование мероприятий
- Разграничение ролей (администратор, вожатый, медсестра)

## Структура базы данных
- `Groups` — отряды
- `Children` — дети
- `Users` — пользователи системы
- `HealthRequests` — медицинские заявки
- `Events` — мероприятия
